Install Beat Saber on Meta Quest

  1. Install ZArchiver on your device.
  2. Open ZArchiver and extract the ZIP file.
  3. Open the folder that was extracted.
  4. Install the Beat Saber APK from inside the folder.
  5. Copy the folder com.beatgames.beatsaber to Android/obb/ on your Meta Quest.
